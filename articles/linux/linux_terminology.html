Linux Terminology

When you look at articles about Linux, you might see some unfamiliar words, such as desktop environment, distro, display manager, etc.

You might be thinking: "what are those?", and then you found this webpage. So here it is, the explanation of the Linux terminology.

Linux Distribution, Linux Distro, or Distro

A Linux distribution, Linux distro, or distro is a variation of an operating system that uses Linux as its kernel.

Desktop Environment or DE

A desktop environment or DE is a suite of applications that consists of the window manager, and other apps related to the desktop environment, such as the panel, dock, terminal emulator, and various other utilities.

Some examples of desktop environments include KDE, GNOME, Cinnamon, Xfce, MATE, and LXQt.

Panel

Taskbar.

Dock

A dock on Linux has no equivalent on Windows, on macOS it is also called a dock.

Window Manager

A window manager is what makes windows appear on your screen. A desktop environment includes a window manager, but adds more applications and utilities. Window managers can be used without a desktop environment, but not vice versa.

Terminal Emulator

Command prompt on Windows, Terminal on macOS.

Display Manager

Login screen.
