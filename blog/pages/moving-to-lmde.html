Moving to LMDE

Recently, Linux Mint Debian Edition 5 was released. It looked interesting, so I gave it a try, liked it, and replaced Ubuntu with it.

Previously, I was using Ubuntu 20.04 LTS, but I wasn't a big fan of it. I tried Debian 11, but decided it was not for me.

Why move to another distro?

Snap packages and GNOME. Seriously.

I was never a big fan of Snap packages. I use Flatpak. I never liked GNOME. I prefer using Xfce or MATE as a desktop environment.

I use Arch, btw. You should use it too.

I prefer using the stable Debian base, so moving to Arch, Manjaro, or any rolling release distro wasn't an option.

Why not [insert distro name here]?

Because I liked Linux Mint and its default configuration just works out-of-the-box.

Do you still use the default Cinnamon desktop?

Yes, I still have it installed and I use it occasionnally.

Why LMDE and not the original Linux Mint?

I liked Debian, I just didn't like configuring it. LMDE ships with most of my preferred settings by default.
