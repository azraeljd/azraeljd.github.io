Linux Package Management

This one day, you decided to try out Linux. You installed it into your computer and voila! You have Linux installed.

But wait, you see that everything looks very outdated. Then you realize, you don't know how to manage your applications on Linux. But fear not, this guide is here to help you.

What is a Package Manager?

A package manager is essentially the thing that you use to download software. The method of downloading software using a package manager is a lot more reliable than downloading some random .exe file from a random website.

Generally, the packages in a Linux distribution's repository are curated by the Linux distribution's maintainers, are open source (unless you download packages from the non-free repository) and are tested by the Linux distribution's QA team before being added to the Linux distribution's package repository.

Package Managers in Linux Distributions

Linux distributions have different package managers. Some of the most popular package managers are:

  • APT in Debian, Ubuntu, and derivatives.
  • Pacman in Arch, Manjaro, and derivatives.
  • DNF in Fedora, Red Hat Enterprise Linux, and derivatives.

Using a Linux Package Manager

Using a Linux package manager is very simple. What you have to do is to determine what Linux distribution you are using. If you are using Ubuntu, Linux Mint, Pop!_OS, or Debian, you will be using the APT package manager. If you are using Arch or Manjaro, you will be using the Pacman package manager.

To use a package manager, you will have to use the terminal.

APT Package Manager

Before you use the APT package manager, it is highly advised to run sudo apt update && sudo apt upgrade to update the package list and upgrade outdated packages.

  • Install packages: sudo apt install <package name>
  • Remove packages: sudo apt remove <package name>
  • Search for a package: apt search <package name>
  • List installed packages: apt list --installed

Pacman Package Manager

Before you use the Pacman package manager, it is highly advised to run sudo pacman -Syu to update the package list and upgrade outdated packages.

  • Install packages: sudo pacman -S <package name>
  • Remove packages: sudo pacman -Rs <package name>
  • Search for a package: sudo pacman -Ss <package name>
  • List installed packages: sudo pacman -Qn
