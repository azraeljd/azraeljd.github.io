azraeljd.github.io Contributing Guidelines

Thank you for your interest in contributing to azraeljd.github.io. Please read these contributing guidelines before you contribute to this website.

About this Website

This website is about free and open source software.

How to Contribute

For Programmers and Web Designers

  • Fixing bugs in the HTML or CSS code.
  • Making a better design for the website.

For writers, people fluent in languages other than English, or non-programmers

  • Translate the website.
  • Write articles.
